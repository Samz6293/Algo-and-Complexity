# Algorithms and Complexity 

Repository for the codes done in Algorithms and Complexity course 66-A.

## 📂 Repository Structure

The repository is divided into three main directories to keep everything organized:

* **`class/`** * This folder contains the code  written during class.
    * It is organized by date (e.g., `21-06-26`, `28-06-26`) so you can easily match the code to your lecture notes for that specific day.
* **`example/`**
    * This folder contains the reference code and examples directly provided by our professor. 
* **`practice/`**
    * These are my personal practice files and experiments.


## 💡 Navigation tips

* **Reviewing a specific lecture?** Head straight to the `class/` folder and look for the date of that lecture.
* **Need the official reference?** Check the `example/` folder for the professor's exact implementation.
* **Want to see how an algorithm is built step-by-step?** Browse through the `practice/` folder to see different ways I've tried putting the logic together.

Good luck with your studying! If you spot any bugs in my practice code or have suggestions for optimization, feel free to let me know.
